# 🚀 Simple CRM System

A full-stack Customer Relationship Management (CRM) application built using the MERN stack (MongoDB, Express, React, Node.js). This project allows users to manage leads, authenticate securely, and perform CRUD operations efficiently.

---

## 📌 Features

-  User Authentication (Register & Login)
-  Dashboard to manage leads
-  Add new leads
-  Edit existing leads
-  Delete leads
-  RESTful API integration
-  MongoDB database storage
-  Improved UI with modern styling

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Axios
- React Router DOM

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB (Local)

## ⚠️ Common Issues

- ❌ MongoDB not connecting → Ensure MongoDB service is running  
- ❌ Modules not found → Run `npm install`  
- ❌ Port already in use → Change port in `.env`
